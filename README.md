# AdventureWorks Business Intelligence Dashboard

## Project Overview
This project is a fully interactive Power BI dashboard built using the AdventureWorks dataset to simulate a real-world business intelligence reporting environment.

The objective was to design a decision-focused analytics solution that allows stakeholders to monitor performance, identify trends, and uncover insights across sales, customers, and products.

---

## Business Questions Answered
- Which products generate the most revenue and profit?
- Where are returns affecting performance?
- Are monthly targets being achieved?
- Which customers drive the highest value?
- How do regional trends compare?

---

## Dashboard Capabilities
- Executive KPI overview for Revenue, Profit, Orders, Return Rate
- Drill-down analysis by category, subcategory, and product
- Monthly performance comparison vs previous month
- Target vs Actual gauge indicators
- Regional performance map filtering
- Customer insights dashboard
- Interactive navigation using buttons and bookmarks
- Dynamic slicers for real-time filtering
- Tooltip analytics pages for deeper category analysis
- Decomposition tree and key influencer visuals for root-cause analysis

---

## Data Model Structure
Implemented a star schema using:

Fact Tables
- Sales Data
- Returns Data

Dimension Tables
- Calendar
- Customer
- Territory
- Product
- Subcategory
- Category

---

## Advanced DAX Measures Created
Examples include:
- Rolling Revenue (10-day, 90-day)
- Adjusted Price / Revenue / Profit
- Return Rate Calculations
- Year-to-Date Revenue
- Previous Month Metrics
- Target Gap Analysis
- Average Revenue per Customer
- Weekend Orders
- High Ticket Orders
